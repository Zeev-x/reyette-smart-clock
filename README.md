# ESP32 Smart Clock Flasher


[Klik Di sini](https://zeev-x.github.io/reyette-smart-clock/) untuk melakukan flashing pada esp32 anda.

## Penyambungan pin
| Module   | ESP32 Pin |
|----------|----------|
| OLED SDA | 21       |
| OLED SCL | 22       |
| BMP SDA  | 21       |
| BMP SCL  | 22       |
| RTC CLK  | 13       |
| RTC DAT  | 12       |
| RTC RST  | 14       |
| SD CS    | 5        |
| SD MOSI  | 23       |
| SD CLK   | 18       |
| SD MISO  | 19       |
| Button   | 0        |
